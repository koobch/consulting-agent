## Phase 0: 리서치 플랜 수립

- [ ] 비교 대상 기업 목록이 정해졌는가?
- [ ] 투입 에이전트 순서가 확정되었는가?
- [ ] `knowledge/self-improvement-log.md`의 과거 EP 패턴을 확인했는가?

## Phase 0.5: 팩트시트 확보 (Company Factsheet) — 최우선
- [ ] 대상 기업의 전체 제품/서비스 목록이 domain-profile의 확인 소스에서 직접 확인되었는가?
- [ ] 각 제품의 지원 플랫폼이 실제 스토어 페이지에서 확인되었는가? (추측 금지)
- [ ] **각 제품의 정확한 출시일(YYYY.MM.DD)이 기재되었는가?** (EP-011)
- [ ] **종료/중단된 제품이 빠짐없이 기재되었는가?** ("[제품명] 서비스 종료/중단" 뉴스 검색) (EP-011)
- [ ] **각 제품의 수익 모델(domain-profile 참조)이 확인되었는가?** (EP-010)
- [ ] **유통 주체-개발·생산 주체 관계와 매출 인식 구조가 명시되었는가?** (EP-012)
- [ ] 한국 기업 재무/직원 데이터가 DART API에서 직접 조회되었는가? (fs_div=OFS)
- [ ] 해외 기업 재무 데이터가 SEC EDGAR / TDnet / HKEX에서 직접 조회되었는가?
- [ ] 경쟁사/비교 대상 기업도 동일 수준의 팩트시트가 작성되었는가?
- [ ] fact-checker 실시간 검증을 통과했는가?
- [ ] 모든 재무 수치에 엔터티 라벨(`[그룹]`/`[별도]`)이 부여되어 있는가?
- [ ] **역량 이력(전 제품, 카테고리별/채널별 경험, 성공/실패 패턴)이 수집되었는가?** (EP-013)
- [ ] **전신 회사 시절 제품이 "레거시/연혁"으로 분리되어 있는가?** (EP-014: 10년+ 전 제품은 현재 역량으로 인용 금지)
- [ ] **모든 제품의 실제 개발·생산 주체가 확인되었는가?** (EP-015: 유통 주체 ≠ 개발·생산 주체, 자회사 간 제품 귀속 혼동 방지)
- [ ] **DART 분기보고서의 분기별 매출 추이가 제품 출시 타이밍과 대조되었는가?**
- [ ] **사용자가 팩트시트를 검토하고 승인했는가?** ← 게이트 #1

## Phase 1: 가설 수립
- [ ] 팩트시트를 읽은 후에 가설을 수립했는가? (팩트시트 없이 시작 금지)
- [ ] 핵심 질문이 명확하고 의사결정에 연결되는가?
- [ ] 가설이 3~7개이며 MECE인가?
- [ ] 각 가설이 검증 가능하고 구체적인가?
- [ ] 가설이 팩트시트의 사실과 모순되지 않는가?
- [ ] 가설의 확인/반박 모두 의미 있는 시사점을 제공하는가?

## Phase 2: 프레임워크 설계
- [ ] 프레임워크가 케이스에 적합한가?
- [ ] MECE 분해가 2~3 레벨까지 완료되었는가?
- [ ] 각 말단 노드에 구체적 분석 질문이 있는가?
- [ ] 가설과 분석 항목이 매핑되어 있는가?
- [ ] 데이터 수집 계획이 현실적인가?
- [ ] **전략/성장 리서치에서 내부 역량 분석 축이 프레임워크에 포함되었는가?** (EP-013)
- [ ] **목표 시장/세그먼트의 성장률 분석 항목이 포함되었는가?** (EP-016)
- [ ] **(Standard/Deep) 사용자가 프레임워크를 확인했는가?** ← 게이트 #1.5

## Phase 3: 데이터 수집 + 실시간 검증 루프
- [ ] 프레임워크의 모든 항목에 대해 수집을 시도했는가?
- [ ] 핵심 수치에 최소 2개 이상 독립 소스가 있는가?
- [ ] 모든 데이터에 출처 URL과 기준 시점이 있는가?
- [ ] 신뢰도 등급이 부여되어 있는가?
- [ ] 데이터 갭이 솔직하게 명시되어 있는가?
- [ ] 다국어 검색을 시도했는가?
- [ ] 한국 기업 재무 데이터에 DART API를 1차 소스로 사용했는가?
- [ ] 해외 기업 재무 데이터에 SEC EDGAR/TDnet/HKEX를 사용했는가?
- [ ] 엔터티(그룹/자회사) 데이터가 명확히 분리되어 있는가?
- [ ] **각 데이터 배치가 fact-checker 실시간 검증을 통과했는가?** ← 핵심
- [ ] **사용자가 데이터 수집 결과를 확인했는가?** ← 게이트 #2

## Phase 4: 종합 팩트체크
- [ ] Level 0 (서술적 주장) 전수 검증 완료?
- [ ] Level 3 (수치 정확성) 교차 검증 완료?
- [ ] Level 4 (논리 검증) 일관성 확인 완료?
- [ ] 핵심 수치의 교차 검증이 완료되었는가?
- [ ] 출처의 신뢰성이 평가되었는가?
- [ ] 논리적 비약이 없는가?
- [ ] 상식적 검증(Sanity Check)을 통과하는가?
- [ ] 통화/단위/시점의 일관성이 확인되었는가?
- [ ] 회계연도(FY) vs 달력연도(CY) 구분이 정확한가?
- [ ] 엔터티 혼동이 없는가?
- [ ] ❌ False 또는 🔴 Disputed 항목이 0건인가?

## Phase 5: 인사이트 도출
- [ ] 팩트시트·검증 데이터·가설이 모두 준비되었는가? (미충족 시 작업 거부)
- [ ] 모든 인사이트가 데이터에 근거하는가?
- [ ] So What이 명확하고 실행 가능한가?
- [ ] 가설 검증 결과가 명확한가?
- [ ] 반론(Devil's Advocate)을 고려했는가?
- [ ] 상관관계와 인과관계를 구분했는가?
- [ ] 전략적 시사점이 단기/중기/장기로 구분되었는가?
- [ ] 인사이트가 팩트시트와 모순되지 않는가?
- [ ] **매출 전망에서 수익 모델별 매출 커브가 올바르게 적용되었는가?** (수익 모델별 차이 확인 (domain-profile 참조)) (EP-010)
- [ ] **분석 대상 엔터티 별도 매출과 총매출(Gross)이 혼동되지 않았는가?** (EP-012)
- [ ] **전략 제안이 기업의 제품의 역량 이력에 근거하는가?** (EP-013)
- [ ] **역량 갭이 있는 제안에 리스크/전제조건이 명시되었는가?** (EP-013)
- [ ] **"이 기업이 실제로 실행할 수 있는가?" 검증을 통과했는가?** (EP-013)
- [ ] **10년 이상 전 역량을 현재 역량으로 인용하지 않았는가?** (EP-014)
- [ ] **전략 목표 시장/세그먼트의 성장률이 확인되고, 축소 시 리스크가 명시되었는가?** (EP-016)
- [ ] **KPI/목표치를 역산했을 때 본문 제품별 합산과 일치하는가?** (EP-019: 외부 기여분 포함 시 명시 필수)
- [ ] **사용자가 인사이트를 검토하고 승인했는가?** ← 게이트 #3
